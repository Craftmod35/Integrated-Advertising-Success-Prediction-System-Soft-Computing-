# Integrated Advertising Success Prediction System (Soft Computing)

This project is a comprehensive **Decision Support System** developed as part of the Soft Computing course at **Mansoura University**.

## 🚀 Key Features

- **Phase 1: Customer Segmentation (SOM)**: Using Self-Organizing Maps to identify suspect/fraud applications.
- **Phase 2: Sales Prediction (ANN)**: A Deep Learning model (97% R2 Score) to forecast future sales.
- **Phase 3: Decision Logic (Fuzzy)**: A Fuzzy Logic controller that converts numbers into investment decisions.
- **Reinforcement Learning**: Strategic pathfinding for market engagement.

## 🛠 Tech Stack

- **Frameworks**: TensorFlow, Scikit-Fuzzy, MiniSom.
- **Environment**: Developed using Python 3.12.
